Beta Documentation

Extends: ArrayGrid

Description

The EPGGrid node class implements an Electronic Program Guide (EPG) component. The channels are represented as horizontal rows, one for each channel. Each row has a channel name on the left, and an array of programs on that channel to the right. The size of each program depends on its duration. One of these programs has a remote control focus highlight indicator on it, and this highlight can be moved around using the remote control (as long as the EPGGrid node has remote control focus).

Fields

Field Type Default Use
content ContentNode none Specifies the content for the list. See Data Bindings below for more details.
leftEdgeTargetTime time none

This is the time that the left edge of the EPG grid either represents, or is in the process of animating to represent. Setting it causes the EPG grid to animate to that time. The field value is initially set such that the now bar is as close to the left edge of the EPG grid as allowed by rules of the minimumNowBarOffset field value. The field value is ignored when the grid is in Now/Next mode (see the nowNextMode field).
contentStartTime time 0 The earliest time that the EPG grid can be moved to
duration double

9000

 Width (in seconds) of the visible section of the EPG grid. As an example, setting it to 10800 will cause the EPG grid to display 3 hours worth of programs horizontally.
autoDismissTime time 0 Specified the time (in seconds) that the EPG grid will be displayed before automatically hiding itself. Setting it to 0 will cause the EPG grid to never hide itself.
minimumNowBarOffset time 300 Specifies (in seconds) the nearest that the Now bar (bar indicating the current time) can be to the left-hand edge of the EPG grid. If the Now bar advances to the minimumNowBarOffset field value plus 30 minutes, the EPG grid will automatically scroll right 30 minutes.
nowNextMode Boolean false When set to true, the EPG grid displays two vertical columns of programs, showing Now and Next for each channel
useTunerTime Boolean false When set to true, and the time derived from the tuner is available, then the EPGGrid node will use the time derived from the tuner. Otherwise the system time will be used.
maxDays integer 7 Specifies the total width of the EPG grid in days
homeChannel integer 0 This represents the index of the channel that is the home channel. The intended use is to be the channel that was being watched when the EPG grid was called up. Pressing Rewind on the remote control when the grid is all the way left (that is, Now bar showing or in Now/Next mode) will cause the grid to focus on this channel. Index 0 is the first channel in the grid.
programFocused integer 0 The program number of the focused program (within the focused channel)
programUnfocused integer 0 The program number of the last program to have the focus (the program that just lost the focus)
jumpToProgram integer 0 Set this field to the program index of the program you wish to have focus (in the focused channel). The grid will immediately jump to and focus on this program.
focusBitmapUri URI string ""

Specifies the bitmap file used for the focus indicator when the list has focus. In most cases, this should be a 9-patch image that specifies expandable regions. Only set this field to specify a custom bitmap that differs in appearance from the default bitmap.
nowBarBitmapUri URI string ""

Specifies the bitmap file used for the Now Bar (a vertical bar indicating the current time). In most cases, this should be a 9-patch image that specifies expandable regions. Only set this field to specify a custom bitmap that differs in appearance from the default bitmap.

channelInfoColumnLabel string ""

Specifies text used as a label for the channel info column (the left most column).  If this is empty the label will be set to the day of the week that the left-hand edge of the EPGGrid represents.
channelInfoBackgroundBitmapUri URI string ""

Specifies the bitmap file to use as the background for the channel names in the grid. In most cases, this should be a 9-patch image that specifies expandable regions. Only set this field to specify a custom bitmap that differs in appearance from the default bitmap.

programBackgroundBitmapUri URI string ""

Specifies the bitmap file to use as the background for the program names in the grid. In most cases, this should be a 9-patch image that specifies expandable regions. Only set this field to specify a custom bitmap that differs in appearance from the default bitmap.

timeBarBitmapUri URI string ""

Specifies the bitmap file to use as the background of the time bar that appears above the EPG channel grid. In most cases, this should be a 9-patch image that specifies expandable regions. Only set this field to specify a custom bitmap that differs in appearance from the default bitmap.
overlayBitmapUri URI string ""

Specifies an overlay that is drawn on top of the entire EPG. This is used by default to make the bottom visible rows of the EPG grid appear to fade out, but could be used to add any overlay be drawn on top of the EPG grid. In most cases, this should be a 9-patch image that specifies expandable regions. Only set this field to specify a custom bitmap that differs in appearance from the default bitmap.

The overlay bitmap is drawn over the full width of the EPG grid with its bottom aligned to the bottom of the EPG grid. If the overlayHeight field value is greater than 0, the image will be stretched to the specified height (that is, from the bottom of the EPG grid to the specified height). If overlayHeight field value is set to 0, the bitmap will be drawn at its default height.
overlayHeight float 0 Specifies the height of the overlay image (see the overlayBitmapUri field). If set to 0 the overlay image is drawn at its default height.
channelInfoWidth float 189 Width of the column showing the channel names in the EPG grid.
infoGridGap float 0 Spacing between the channel names column and the main grid.
channelInfoFont font system default Specifies the font for the channel names
programTitleFont font system default Specifies the font for the program titles
timeLabelFont font system default Specifies the font for the time labels that are drawn along the top of the EPG
channelInfoTextColor color 0xffffff99 Specifies the text color for the channel names
programTitleColor color 0xffffffff Specifies the text color for program titles when the program is unfocused
programTitleFocusedColor color 0xff Specifies the text color for program titles when the program is focused
programNowNextTimeColor color 0xffffff99 When nowNextMode is true, specifies the text color for the time that appears in the now/next program cells when the program is unfocused
programNowNextTimeFocusedColor color 0x99 When nowNextMode is true, specifies the text color for the time that appears in the Now/Next program cells when the program is focused
timeLabelColor color 0xffffff99 Specifies the text color for the time labels that are drawn along the top of the EPG
timeLabelOffset float 14 Specifies the offset from the edge of the grid of the leftmost time label
timeBarHeight float 50 Specifies the height of the region above the channel grid where the time labels are displayed.
nowBarWidth float 4 Specifies the width of the Now bar
nowBarHeight float 410 Specifies the height of the Now bar
programHorizMargin float 14 Specifies the width of left and right margins of program cell text.
programNowNextHorizMargin float 14 Specifies the width of left and right margins of program cell text in Now/Next mode.
programNowNextTimeTitleGap float 15 Specifies the gap between the Now and Next entries

Data Bindings

The data model for the EPGGrid node should have a single ContentNode node as the root node. This node should have one child per channel in the grid. Each child channel should have one child per program for that channel.

The programs within the channel must be sorted by start time, earliest first. There should be no gaps between programs.

Channel attributes

Attribute Type Description
TITLE string The name of this channel, to be visible only if HDSMALLICONURL is not set or if the icon failed to load
HDSMALLICONURL URI string The image file for the channel logo

Program Attributes

The following attributes are available for each program:

Attribute Type Description
TITLE string Program title
PLAYSTART time Start time for this program.
PLAYDURATION time Duration of  this program
HDSMALLICONURL URI string The image file for an icon that appears beside the program title (optional)
