roDataGramSocket

Skip to end of metadata
  • Created by Jeff Bush, last modified by Kevin Cooper on Jul 05, 2015
Go to start of metadata

The roDataGramSocket component enables Brightscript apps to send and receive UDP packets. The interface is modeled on and works much like standard Berkeley sockets.

Supported Interfaces

  • ifSocket
  • ifSocketAsync
  • ifSocketStatus
  • ifSocketOption
  • ifSocketCastOption

Note: some firmware versions may implement ifSocketCastOption as ifSocketCast.

Supported Events

  • roSocketEvent

Description

This object is created without any arguments:

  • CreateObject("roDataGramSocket")
