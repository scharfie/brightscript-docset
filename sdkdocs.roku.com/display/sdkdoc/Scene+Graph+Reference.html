The following provides complete reference information on all node classes, component XML file elements, component functions, and BrightScript components and interfaces, available in the Roku Scene Graph API. Each node description includes a table of the fields that may or must be defined in the node, with a description of the possible values for each field. Many of the node descriptions include information on the data bindings of the node, that is, the data required for the node to function. A link is provided at the top of the node description to the parent node or nodes from which certain fields and characteristics are inherited.

The following illustrates the basic class hierarchy of the Scene Graph node classes as derived from the abstract node classes in the API.
